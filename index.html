    LIST OF ACCEPTABLE VALID ID'S

    Below are the accepted requirements for PLDT's online application to ensure that you upload the front and back parts of your ID. Ensure that it is readable and clear to avoid disapproval or a long wait for your application.

    Primary POID

    • Philippine Passport (Front & Back Portion with Signature)
    • Philippine Driver License
    • Unified Multi-Purpose ID
    • PRC/IBP/OFW/DIPLOMA ID (Front & Back Portion)
    • AFP ID
    • PNP ID
    • Philippine Air Force ID
    • Locally-issued Credit Card (either photo or non-photo card)
    • Digitized Postal ID
    • Digitized Philhealth ID
    • Digitized Voter's/Comelec ID with QR Code
    • Firearms License ID (Digitized/Card Type only)
    • Alien Certificate of Registration with Permanent Status(for Foreign Nationals)
    • Police Certificate Card
    • National ID (Front & Back Portion)

    Secondary POID

    • SSS/GSIS/TIN with Photo
    • Voter's ID (Thumbmark only)
    • Senior Citizen's ID
    • Non-digitized Postal ID
    • Non-digitizzed Philhealth ID
    • Company ID (not included in the Top 1000 Corporations)
    • Barangay ID
    • Student ID (Top 228 Universities and Colleges)
    • Company ID (Top 1000 Corporations)

    Foreign Applicant

    • Alien Certificate of Registration (ACR) with Permanent Status
    • Special Retirees Resident Visa (SRRV)
    • Alien Employment Permit (AEP)
    • Certified Diplomatic ID (for Diplomats/Embassy Staff)
